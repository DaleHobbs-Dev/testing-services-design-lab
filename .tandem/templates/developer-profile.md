# Template: Developer Profile

This template is a starting point for your personal developer profile.

Tandem uses this profile to adjust:

- explanation depth
- how much background context to include
- how challenging understanding checks should be
- how much it should assume you already know

You would normally copy this into `~/.claude/CLAUDE.md` under a `## Developer Profile` heading and personalize it there.

Keep it honest and practical. This works best when it reflects how you actually work right now, not the version of you that sounds most impressive.
